Internship Task 2

Building a Professional Web Workflow using Gulp & Nunjucks

Project Overview

This project demonstrates how modern websites are built using reusable templates and automated build tools instead of manually managing HTML files.

Why This Project Matters

  • Reduces repetition using reusable templates
  • Improves productivity through automation
  • Ensures consistency across pages

Technologies & Tools Used

  • HTML5 – Structure of web pages
  • CSS3 – Styling and layout
  • Nunjucks – Template engine
  • Gulp – Task automation
  • Node.js – Build environment

Learning Outcomes

This task helped me understand real-world web development workflows, asset management, and industry best practices.

Project Highlights

Reusable Templates

Nunjucks allows building reusable layouts, reducing duplication and improving consistency across pages.

Automated Build

Gulp automates tasks like compiling templates and managing assets, saving time and avoiding manual errors.

Industry Workflow

This project follows real-world development practices used in professional web development teams.
